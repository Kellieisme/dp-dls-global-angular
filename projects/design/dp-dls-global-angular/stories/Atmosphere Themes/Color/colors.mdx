  # Color Tokens for light and dark mode

      Upgrading ```@jeppesen-foreflight/dp-dls-global-angular``` and ```@jeppesen-foreflight/dls-global-assets```

      ### For Angular 18 projects:
      ```json
      {
        "dependencies": {
          "@jeppesen-foreflight/dp-dls-global-angular": "^2.5.1",
          "@jeppesen-foreflight/dls-global-assets": "^2.0.0"
        }
      }
      ```

      ### For angular 19 projects:
      ```json
      {
        "dependencies": {
          "@jeppesen-foreflight/dp-dls-global-angular": "^3.4.0",
          "@jeppesen-foreflight/dls-global-assets": "^2.1.2"
        }
      }
      ```


  The atmosphere theme is controlled by the assets library. This library exposes some variables that you can use so that you can adhere to the theme and seamlessly maintain it in both light and dark mode. Use these variables in your scss to maintain light and dark mode in code that is not part of the atmosphere library but needs atmosphere theme-ing

      See below a example of how your scss/sass code would look.
      ```scss
      div {
          background:var(--foundation-ui-background-lowest)
      }
      ```

  The library is by default in dark mode , however if you wish to hard set it to light you can add `theme-light` to the `body` tag.

  In index.html

  ```html
  <body class="theme-light">
    <app-root></app-root>
  </body>
  ```
