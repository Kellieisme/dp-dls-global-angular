    # Glass Effects

    To implement glass effect on a surface you can use mixins provided by Atmosphere's assets package.

    Note: To use the composition Mixins you must import the following in your scss file.
    ```scss
     @use "@jeppesen-foreflight/dls-global-assets/dist/scss/base/_compositionMixins.scss" as composition;
    ```


    Below is an example of how to implement glass effects on a surface.
       <Canvas of={GlassStories.Glass} />



    In your scss file:

    ```css
    @use "@jeppesen-foreflight/dls-global-assets/dist/scss/base/_compositionMixins.scss" as
    composition;

    .glass {
    width: 300px;
    height: 300px;
    z-index: 100;
    position: fixed;
    top: 0px;
    @include composition.composition-dark-ui-glass-background-blur;

    .theme-light & {
        @include composition.composition-light-ui-glass-background-blur;
    }
    }
    ```

    In the corresponding html file:

    ```html
    <div class="glass">This is a glass sheet</div>
    ```
