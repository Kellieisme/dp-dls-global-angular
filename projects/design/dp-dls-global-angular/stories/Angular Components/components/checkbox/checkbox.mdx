# Checkbox

## API References

The Checkbox is a direct implementation of the Angular Material Checkbox component. Custom styles are applied using the Angular Material theming API and additional CSS rules.

### Selectors

`mat-checkbox`

### Usage

```typescript
import { MatCheckboxModule } from "@angular/material/checkbox";


@Component({
  selector: 'checkbox-overview-example',
  standalone:true,
  imports: [MatCheckboxModule]
  })
```

### Style Classes

<Markdown>
  {`
| Name                                                       | Description                                                                                  |
| ---------------------------------------------------------- | -------------------------------------------------------------------------------------------- |
| error                                                      | Adds styles for error state. |
`}
</Markdown>

### Properties

<Markdown>
  {`
| Name                                                       | Description                                                                                  |
| ---------------------------------------------------------- | -------------------------------------------------------------------------------------------- |
| @Input() id: string                                        | A unique id for the checkbox input. If none is supplied, it will be auto-generated.|
| @Input() disabled: boolean                                 | Disables checkbox. Defaults to true. |
| @Input() checked : boolean                                 | Sets checkbox to checked state. Defaults to true. |
| @Input() indeterminate : boolean                           | Sets checkbox to partially complete state. Defaults to true. |
| @Output() change: EventEmitter<MatCheckboxChange>          | Event emitted when the checkbox's checked value changes.|
| @Output() indeterminateChange: EventEmitter<boolean>       | Event emitted when the checkbox's indeterminate value changes|

`}

</Markdown>
