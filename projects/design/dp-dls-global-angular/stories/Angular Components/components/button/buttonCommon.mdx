# Button

### API reference

The Button (Common) is a direct implementation of the Angular Material Button component. Custom styles are applied using the Angular Material theming API and additional CSS rules.

### Selectors

**Recommended:** `button[matButton]` `button[matButton="outlined"]` `button[matButton="filled"]` `button[matIconButton]`

**Legacy (for backward compatibility):** `button[mat-button]` `button[mat-stroked-button]` `button[mat-flat-button]` `button[mat-icon-button]`

<Red_text>NOTE: The fab and tonal buttons are not supported. </Red_text>

### Usage

In your typescript file.

```typescript
import { MatButtonModule } from "@angular/material/button";
import { MatIconModule } from '@angular/material/icon';
import { IconRegistryModule } from "@jeppesen-foreflight/dp-dls-global-angular/icon-registry";


@Component({
selector: 'button-overview-example',
standalone:true,
imports: [MatButtonModule, MatIconModule, TitleCasePipe, IconRegistryModule]
})
```

Note: To use Icons in the Button Component you need to import the IconRegistryModule.
Make sure your `angular.json` has the path to `svgSet.svg`. You can find the steps to set it up [here](?path=/docs/get-started-install-configure-atmosphere-setting-global-styles-and-fonts--docs).

### Properties

<Markdown>
{`
| Name                                                                     | Description                                                                                  |
| -------------------------------------------------------------------------| -------------------------------------------------------------------------------------------- |
| matButton, matButton="outlined", matButton="filled", matIconButton      | Determines the type of button. Use these directives for new implementations.                 |
| mat-button, mat-stroked-button, mat-flat-button, mat-icon-button        | Legacy directives. Supported for backward compatibility.                                     |
| @Input() disabled: boolean                                               | Disables button. Defaults to false.                                                          |
| @Input() color                                                           | Sets color to "primary" or "accent" for common buttons; "primary", "accent", or "warn" for icon buttons. Default is "primary" |
`}

</Markdown>
