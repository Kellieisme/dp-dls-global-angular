# Button

### API reference

The Button (Common) is a direct implementation of the Angular Material Button component. Custom styles are applied using the Angular Material theming API and additional CSS rules.

### Selectors

`button[mat-button]` `button[mat-flat-button]` `button[mat-stroked-button]` `button[mat-icon-button]`

<Red_text>Note: fab button and tonal button is not supported by Atmosphere </Red_text>
### Usage

In your typescript file.

```typescript
import { MatButtonModule } from "@angular/material/button";
import { MatIconModule } from '@angular/material/icon';
import { IconRegistryModule } from "@jeppesen-foreflight/dp-dls-global-angular/icon-registry";


@Component({
selector: 'button-overview-example',
standalone:true,
imports: [MatButtonModule, MatIconModule, TitleCasePipe, IconRegistryModule]
})
```

Note: To use Icons in the Button Component you need to import the IconRegistryModule.
Make sure your `angular.json` has the path to `svgSet.svg`. You can find the steps to set it up [here](?path=/docs/get-started-install-configure-atmosphere-setting-global-styles-and-fonts--docs).

### Properties

<Markdown>
{`
| Name                                                                     | Description                                                                                  |
| -------------------------------------------------------------------------| -------------------------------------------------------------------------------------------- |
| {type} mat-button, mat-raised-button , mat-flat-button, mat-stroked-button,mat-icon-button | Determines the type of button.|
| @Input() disabled: boolean                                               | Disables button. Defaults to true. |
| @Input() color                                                           | Sets color to "primary" , "accent" or "warn". Default is "primary" |
`}

</Markdown>
