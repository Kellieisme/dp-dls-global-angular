# Theme Toggle

### API reference

The Theme Toggle is a custom component provided by Atmosphere Angular Library to switch between Light and Dark Themes.This component is based on the `ThemeToggleService`


### Selectors

`ba-theme-toggle`

### Usage

In your typescript file.

```typescript

import { Component } from '@angular/core';
import { CommonModule } from '@angular/common';
import {
  ThemeToggleComponent,
  ThemeToggleModule,
  ThemeToggleService,
} from '@jeppesen-foreflight/dp-dls-global-angular/theme-toggle';


@Component({
  selector: 'theme-toggle-example-page',
  standalone: true,
  imports: [
    CommonModule,
    ThemeToggleModule,
  ],
  template: `<ba-theme-toggle></ba-theme-toggle>`,
})
export class CustomThemeToggleComponent { }

```

# Theme Toggle Service

The `ThemeToggleService` provides light and dark theme switching functionality by toggling `theme-light` and `theme-dark` classes on the `document.body`. Components can reactively update based on the active theme using the exposed observable.

### Features

- Apply theme class to `<body>` to control global styles.
- Emits theme changes via `themeChanged$`.
- Exposes `toggleTheme()` and `getCurrentTheme()` for manual control.

### API

`toggleTheme(): void`

Toggles between light and dark themes. Automatically applies corresponding class to `document.body`.

`getCurrentTheme(): Theme`

Returns the currently active theme. Useful for conditionally rendering elements based on the current mode.

`themeChanged$: Observable<Theme>`

An observable that emits the new theme value on every change. Suitable for subscribing in components to react to theme switches.
