# Switch (Slide toggle)

### API References

The Switch (Slide toggle) is a direct implementation of the Angular Material Slide toggle component. Custom styles are applied using the Angular Material theming API and additional CSS rules. Atmosphere enforces usage with defined design tokens.

### Selectors

`mat-slide-toggle`

### Usage

Ensure you import “MatSlideToggleModule” from Angular Material in your typescript code.

```typescript
import { Component } from "@angular/core";
import { MatSlideToggleModule } from "@angular/material/slide-toggle";

@Component({
  selector: "slide-toggle-page",
  standalone: true,
  imports: [MatSlideToggleModule],
  templateUrl: "./slide-toggle.component.html",
  styleUrl: "./slide-toggle.component.scss",
})
export class SwitchPageComponent {}
```

```html
<p><mat-slide-toggle color="warn">Warn</mat-slide-toggle></p>
<p><mat-slide-toggle color="primary">Primary</mat-slide-toggle></p>
<p>
  <mat-slide-toggle labelPosition="before"
    >Text before - Default</mat-slide-toggle
  >
</p>
<p>
  <mat-slide-toggle labelPosition="before" color="warn"
    >Text before - Warn</mat-slide-toggle
  >
</p>
<p>
  <mat-slide-toggle labelPosition="before" disabled>Disabled</mat-slide-toggle>
</p>
```

### Properties

Check out how these Properties work using the controls in the component example pages.

<Markdown>
  {`
| Name             | Type     | Description                                      |
|------------------|----------|--------------------------------------------------|
| @Input() color            | string   | Theme color: 'primary', 'warn'         |
| @Input() disabled         | boolean  | Disables the toggle                              |
| @Input() labelPosition    | string   | Position of label: 'before' or 'after'           |
| @Input() checked: boolean | boolean  | Whether the slide-toggle element is checked or not.                             |
| @Input() required: boolean| boolean  | Whether the slide-toggle is required.                            |
| @Input() hideIcon: boolean | boolean  | Whether to hide the icon inside of the slide toggle.                           |
| @Output() change          |EventEmitter<MatSlideToggleChange>    | An event will be dispatched each time the slide-toggle changes its value.  |
| @Output() toggleChange    |EventEmitter<Void>    | An event will be dispatched each time the slide-toggle input is toggled. This event is always emitted when the user toggles the slide toggle, but this does not mean the slide toggle's value has changed. |
`}
</Markdown>

### Troubleshooting

<Markdown>
  {`
- Make sure to import MatSlideToggleModule in your Angular module.
- Ensure you pass valid values for color (primary, warn).
- LabelPosition accepts only 'before' or 'after'.
- Some themes might override default styling if global tokens are misconfigured.
`}
</Markdown>
