# Radio Button

### API Reference

The radio button is a direct implementation of the Angular Material `mat-radio-button`.  
It is styled using the Boeing Atmosphere theming system through SCSS variables and global typography.  
Use `mat-radio-group` to group options.

<Markdown>
{`
| Selector         | Description                               |
|------------------|-------------------------------------------|
| mat-radio-button | Individual radio option                   |
| mat-radio-group  | Container that groups radio buttons       |
`}
</Markdown>

### Usage

#### Typescript

```ts
import { MatRadioModule } from '@angular/material/radio';
Template
html
Copy
Edit
<mat-radio-group [(ngModel)]="selectedOption">
  <mat-radio-button *ngFor="let opt of options" [value]="opt">{{ opt }}</mat-radio-button>
</mat-radio-group>
```
### Properties

<Markdown>
{`
| Name              | Description                                                                 |
|-------------------|-----------------------------------------------------------------------------|
| @Input() value    | The value assigned to the radio button                                      |
| @Input() disabled | Whether the radio button is disabled                                        |
| @Input() name     | Name of the group (useful when using without \`mat-radio-group\`)           |
| @Input() color    | Theme color: 'primary' | 'accent' | 'warn' (default is 'primary')            |
| @Output() change  | Event emitted when the user selects this radio button                      |
`}
</Markdown>

Source: [Angular Material Radio API](https://v18.material.angular.io/components/radio/api)

### Troubleshooting

- **Missing Imports**: Ensure `MatRadioModule` is imported in your Angular module.
- **No Selection**: Remember to bind `[(ngModel)]` or use reactive forms with `formControlName`.
- **Accessibility**: `mat-radio-group` automatically sets `role="radiogroup"` for screen readers.
