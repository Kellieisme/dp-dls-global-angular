# Text Field (Input)

The Text field is a direct implementation of the Angular Material Input component. Custom styles are applied using the Angular Material theming API and additional CSS rules.

Stories for Text field and Text area both implement the Angular Material Input component. Seperate component stories for Text field and Text area have been created to align with the DLS design library.

`}
