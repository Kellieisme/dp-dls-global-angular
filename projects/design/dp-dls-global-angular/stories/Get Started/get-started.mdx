# Get started

This Storybook aligns to the current release of the Atmosphere Global Angular library.
- [Github - code repository](https://github.com/jeppesen-foreflight/dp-dls-global-angular)

## Components

The library contains both custom components (each with their own entry point) and a style package that themes a
number of Angular Material components.

## Custom

Components in the Custom folder each have their own documentation page generated with Compodoc.

## Material

Components in the Material folder are direct implementations of Angular Material components. Custom styles are applied using the Angular Material theming API and additional CSS rules.

The documentation page for each Material component will describe any special use cases, such as available custom classes.

For API reference, the documentation page will link to the corresponding Angular Material documentation page.

## Patterns

Patterns are Storybook stories that demonstrate additional usage patterns combining multiple components to construct a layout.
