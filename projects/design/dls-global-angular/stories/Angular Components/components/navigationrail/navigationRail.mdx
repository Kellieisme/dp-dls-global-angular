# Navigation Rail

The `NavigationRailComponent` provides a side navigation experience with customizable branding, menu items, and collapsible behavior.

## Features

- Customizable branding (app name & logo)
- Supports different presentation styles: `Default`, `Standalone`, and `Modal`
- Configurable expansion and collapse behavior
- Slot for user profile and notifications button

## API Reference

### Selectors

`ba-navigation-rail`

### Properties

<Markdown>
{`
| Name                        | Type    | Description                               |
|-----------------------------|---------|-------------------------------------------|
| @Input() hideLabels         | boolean | Hide labels for menu items                |
| @Input() wide               | boolean | Enable wide mode for the navigation rail  |
| @Input() appLogo            | string  | Path to application logo                  |
| @Input() homeRoute          | string  | Route for home navigation                 |
| @Input() menuItems          | Array   | List of navigation menu items             |
`}
</Markdown>
