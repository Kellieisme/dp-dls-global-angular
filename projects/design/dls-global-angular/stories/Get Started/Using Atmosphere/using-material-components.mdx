# Using Material Components

> Angular Material components can be used according to the [Angular Material documentation](https://v17.material.angular.io/components/categories).
> If the stylesheets are included in the angular.json as per the [configuration instructions](?path=/docs/get-started-install-configure-atmosphere-setting-global-styles-and-fonts--docs), the components will be styled without any further action needed.

> The theme-ing for atmosphere components are completely handled internally by the Library.
> To set up the toggle option for light and dark mode you can checkout [Light/Dark Toggle Module](?path=/docs/atmosphere-angular-components-components-custom-theme-toggle--docs).
