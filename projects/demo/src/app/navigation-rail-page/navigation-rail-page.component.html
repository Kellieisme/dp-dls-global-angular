Navigation Rail Component

The navigation rail is displayed on the left side of this application.

Features

  • Slim vertical navigation (80px narrow, 100px wide)
  • Icon-based navigation with optional labels
  • Support for up to 4 bottom components (user profile, notifications, etc.)
  • Integrated with Angular Router
  • Theme-aware (light/dark mode support)

Configuration

The navigation rail in this demo is configured with:

  • Wide mode: 100px width
  • Labels visible: Menu item labels shown
  • App logo: Boeing logomark
  • Bottom component: User profile with menu
